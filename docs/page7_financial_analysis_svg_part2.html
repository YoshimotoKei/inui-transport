財務分析：業績推移と財務健全性・成長性の評価 (2)

財務健全性の評価

乾汽船の財務健全性は極めて高い水準にあります。2024年3月期末の自己資本比率は47.5%と業界平均を上回っており、純資産残高は316.18億円を確保しています。また、現金及び同等物は139.80億円と潤沢な手元流動性を維持し、ネットキャッシュポジションを実現しています。キャッシュフローについては、2024年3月期は投資活動や財務活動でのマイナスが大きくなっていますが、これは将来の成長に向けた戦略的な投資と配当などの株主還元によるものです。

成長性の評価

乾汽船の成長性は、海運市況に大きく左右されますが、不動産事業による安定収益がボラティリティを抑制しています。今後の成長要因としては、新造ハンディバルカーの投入による燃費性能や積載効率の改善、および不動産再開発完了後の賃料収入増が期待されます。一方、地政学リスクの顕在化や世界不況による海運需要の冷え込みがリスク要因となります。総合的に見れば、乾汽船は財務健全性が極めて高く、市況好転時には大きな成長余地があると評価できます。

総合評価

乾汽船の財務分析を総合すると、極めて高い財務健全性と市況に応じた成長余地を持つ企業と評価できます。自己資本比率約50%、ネットD/Eレシオ実質ゼロ近辺という強固な財務基盤は、海運業界の市況変動に対する耐性を高めています。また、不動産事業による安定収益が全体の下支えとなり、海運市況の変動に対するバッファーとして機能しています。今後は、新造船投入や不動産再開発の完了により、中長期的な成長基盤がさらに強化されると期待されます。
