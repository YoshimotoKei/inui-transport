競合分析：主要プレイヤー、市場シェア、ビジネスモデル比較

バルク船市場の主要プレイヤー

バルク船市場には、日本の大手海運会社（日本郵船、商船三井など）から中堅の乾汽船、海外のギリシャ系船主、中国系船社、欧州系船社まで、様々なプレイヤーが存在します。日本の大手海運会社は総合海運として多様な船種を運航していますが、乾汽船はハンディサイズバルカーに特化し、オーナーオペレーター戦略を堅持している点が特徴です。

海外では、ギリシャ系船主が伝統的に強い存在感を持ち、市況に応じた機動的な船舶売買を行っています。また、中国系船社は国策との連動や中国の資源輸入需要を背景に急速に成長しています。欧州系船社は環境対応で先行しており、長期契約と市況取引のバランスを重視しています。

ビジネスモデル比較

海運会社のビジネスモデルは、船舶の所有（オーナー）と運航（オペレーター）の軸、および船舶サイズの軸で分類できます。乾汽船は小型船（ハンディサイズバルカー）に特化したオーナーオペレーター戦略を採用しており、「緩やかなOWN主義」と表現される自社船を中心とした船隊構成を維持しています。

一方、日本の大手海運会社は多様な船種・サイズで総合的な海運サービスを提供しており、オペレーター志向が強い傾向にあります。特に川崎汽船は近年「ライトアセット化」を進めています。海外では、ギリシャ系船主がオーナー志向が強く、欧州系船社はオペレーター志向が強い傾向にあります。

乾汽船の競争優位性

乾汽船の競争優位性は、以下の4つの要素から成り立っています：

  1. ハンディサイズバルカーへの特化：小型船ながら多様な港に寄港可能な柔軟性を持ち、多様な貨物を運べる強みがあります。
  2. オーナーオペレーター戦略：「緩やかなOWN主義」による自社船を中心とした船隊構成により、市況変動時にも自律的な運航判断が可能です。
  3. 不動産事業による安定収益基盤：海運市況の変動に左右されない安定した収益源を持ち、事業全体の下支えとなっています。
  4. 健全な財務体質：自己資本比率約50%という高い財務健全性により、市況低迷期にも耐えうる体力を持っています。

乾汽船は「船を持たなければ船会社ではなくなる」という信念に基づき、自社船保有と自律的運航を重視するビジネスモデルを構築しています。これにより、コロナ禍での市況急変時にも素早く対応し、好業績を上げることができました。
