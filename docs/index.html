使用方法

各ページはHTMLファイルとして作成されており、ウェブブラウザで開くことで閲覧できます。SVG形式の図表は、高解像度でスケーラブルな表現が可能で、印刷にも適しています。

色分けについては以下の通りです：

  • 緑色: 強み・ポジティブ要素
  • 黄色: 重要ポイント・注目要素
  • 赤色: リスク・課題
